# 會前Q&A準備｜MeetingPrep

> 圖靈金融｜拜訪前必備工具，30 秒完成會議前準備

---

## 觸發格式

```
開會 {股票代號} {日期}
```

**範例：**
- `開會 3026 4/15`
- `開會 2330 2026-05-20`
- `開會 8103`（日期省略 → 預設今日）

---

## 執行流程

### STEP 1：解析日期格式

接受多種格式，統一轉為 `YYYY-MM-DD`：

| 輸入格式 | 轉換結果 |
|----------|---------|
| `4/15` | `2026-04-15`（補當年） |
| `2026-05-20` | `2026-05-20` |
| （省略） | 今日日期 |

---

### STEP 2：WebSearch 近期新聞

```
搜尋詞："{公司名稱} OR {股票代號} 新聞 site:ctee.com.tw OR site:money.udn.com OR site:cnyes.com"
時間範圍：最近 90 天
```

抓取 3–5 則，整理為：
```python
news_items = [
    {"title": "標題", "source": "工商時報", "date": "2026/04/08"},
    ...
]
```

**⚠️ 必須在執行腳本前完成 WebSearch，將 news_items 以 JSON 格式傳入腳本。**

---

### STEP 3：執行腳本

```bash
python3 /Users/jeffhsu/Desktop/Claude/圖靈金融/scripts/meeting_prep.py \
  "{股票代號}" \
  "{YYYY-MM-DD}" \
  '{news_items_json}'
```

**範例：**
```bash
python3 /Users/jeffhsu/Desktop/Claude/圖靈金融/scripts/meeting_prep.py 3026 2026-04-15 '[
  {"title":"禾伸堂法說會：Q1營收創新高","source":"工商時報","date":"2026/04/03"},
  {"title":"AI 驅動連接器需求，禾伸堂受惠","source":"經濟日報","date":"2026/03/28"}

```

| **頁尾** | Jeff Hsu, CFA｜Turing Financial Group｜機密文件 |

## 建議提問生成邏輯（三大面向，共 8 題）

PDF 第 2 頁分三個區塊，每區塊有色條標示，腳本依財務指標智慧產生：

### 【財務體質】3 題

| 條件 | 自動生成方向 |
|------|------------|
| 毛利率 - 營益率 > 15pp | 費用率偏高，詢問降費計畫與時程 |
| 毛利率 < 20% | 詢問毛利結構改善做法（組合/漲價/降成本）|
| 毛利率 > 40% | 詢問高毛利護城河與潛在風險 |
| 其他 | 詢問原物料/匯率對毛利影響 |
| ROE > 20% | 詢問高報酬可持續性 |
| ROE < 0% | 詢問虧損主因與轉虧為盈里程碑 |
| ROE < 8% | 詢問中期提升股東報酬規劃 |
| 負債比率 > 50% | 詢問短期到期債務與再融資計畫 |
| 負債比率 < 20% | 詢問加槓桿/收購空間 |

### 【產業情況】3 題

| 產業別 | Q1 方向 | Q2 方向 |
|--------|---------|---------|
| 半導體/電子/光電 | AI/封裝訂單能見度 | 關稅/美中供應鏈影響 |
| 生技/醫療 | 法規申請進度 | 競品動態對定價影響 |
| 金融/銀行/保險 | NIM/NPL 最新狀況 | 金融監管新規衝擊 |
| 能源/再生/電力 | 裝機量目標與 IRR | 電價/PPA 談判進展 |
| 零售/餐飲/消費 | SSSG 趨勢 | 展店計畫與成本壓力 |
| 其他 | 產業需求情境判斷 | 差異化優勢與威脅 |

Q3（固定）：今年營收成長驅動力，能見度能看到哪一季？

### 【資本支出】2 題

| 條件 | Q1 方向 |
|------|---------|
| FCF < 0 | Capex 高峰何時結束？現金流轉正前提？ |
| FCF ≥ 0 | 今年 Capex 預算金額與投資方向 |

Q2（固定）：Capex 投入預期回報（產能/效率/市佔），達成 ROI 時程？

## 數據來源優先順序

| 資料 | 優先 | Fallback |
|------|------|---------|
| 財務指標（PE、毛利率等）| five91 `/api/metrics` | yfinance `.info` |
| 公司中文名稱 | five91 `name` | TWSE OpenAPI `t187ap03_L`（公司簡稱）|
| 股價 / 3M漲跌 | yfinance `.TW` / `.TWO` | — |
| 年度財報（營收/毛利/YoY）| yfinance `financials` | — |

> **yfinance fallback 觸發條件**：five91 回傳的 `market_cap` 與 `gross_margin` 均為空時自動啟用。
