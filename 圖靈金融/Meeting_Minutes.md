# Meeting_Minutes.md｜投審會會議紀錄工作規範

> 當使用者提供會議附件、語音檔或逐字稿，並要求撰寫投審會會議紀錄時載入。

---

## 一、任務說明

每次召開投資審議會（投審會）後，使用者將提供：
1. **會議附件**（PDF 或其他資料，如財務報表、申報公告、淨資產報告等）
2. **會議語音檔**（錄音或逐字稿）
3. **上次會議紀錄**（.docx 格式，作為格式範本）

AI 的任務：根據上述資料，撰寫本次投審會的完整會議紀錄 Word 文件（.docx），確保格式、排版、藍色字體範圍完全符合本指令規範。

---

## 二、文件格式規範

### 2.1 整體排版

| 項目 | 規格 |
|------|------|
| 語言 | 繁體中文 |
| 字體 | 全文統一使用 `Kaiti TC`（楷體） |
| 頁面 | A4 尺寸，四邊邊距 1 inch（1440 DXA） |
| 標題第一行 | 圖靈一號併購投資有限合夥（藍色 `#0070C0`，粗體，置中） |
| 標題第二行 | 投資審議會第 N 次會議–會議紀錄（藍色 `#0070C0`，粗體，置中） |

### 2.2 章節層級與縮排

| 層級 | 格式 | 範例 |
|------|------|------|
| 第一層 | 阿拉伯數字加句點 | `1.`、`2.`、`3.`、`4.` |
| 第二層 | 阿拉伯數字加句點（雙碼） | `3.1`、`3.1.1`、`3.2.1` |
| 第三層 | 英文小寫字母加句點 | `a.`、`b.`、`c.` |
| 第四層 | 阿拉伯數字加右括號 | `1)`、`2)`、`3)` |

## 三、藍色字體規範（色碼：#0070C0）

### 3.1 規則總表

| 位置 | 字色 |
|------|------|
| 文件標題（圖靈一號、第 N 次會議） | ✅ 藍色 |
| `1 會議召集`、`2 會議召開` 所有內容 | ❌ 黑色 |
| `3.1 報告事項`（標題字本身） | ❌ 黑色 |
| `3.1.x` 議題標題及以下 `a.`、`1)` 所有內容 | ✅ 藍色 |
| `3.2 討論及決議/承認事項`（標題字本身） | ❌ 黑色 |
| `3.2.x` 議案標題行（「本有限合夥…謹提請討論。」） | ❌ 黑色 |
| `3.2.x` 下的 `a.`、`b.`、決議內文 | ✅ 藍色 |
| `3.3 臨時動議`（標題字本身） | ❌ 黑色 |
| `3.3.1` 臨時動議內容 | ✅ 藍色 |
| `3.4 散會`（標題字本身） | ❌ 黑色 |
| `4 會議附件` 及附件清單 | ❌ 黑色 |

### 3.2 簡單記憶法

> **章節大標題**（`3.1`、`3.2`、`3.3`、`3.4`、`4`）本身為**黑色**；
> 其下的**議題標題與所有子內容**（`3.x.x`、`a.`、`1)`）均為**藍色**。
> 唯一例外：`3.2.x` 的議案標題行（即「本有限合夥…謹提請討論。」）為**黑色**。

## 五、藍色內容填寫規範（每次依資料更新）

### 5.1 報告事項（3.1）

每項報告議題撰寫一個 `3.1.x`，格式：

```
3.1.1 [議題名稱]
   a. [子項目]
      1) [細節]
      2) [細節]
   b. [子項目]
      1) [細節]
```

常見報告類型：
- 投資績效報告（策略概況、績效摘要、利益衝突聲明）
- 法規申報事項（如證交法第 43 條申報）
- 財務報表提報
- 淨資產價值報告
- 銀行現金餘額報告

### 5.2 討論及決議/承認事項（3.2）

每項各自撰寫一個 `3.2.x`，議案標題黑色，內容藍色：

```
3.2.1 [議案標題，黑色]（本有限合夥…謹提請討論。）
   a. [說明計算依據或背景]
   b. [補充說明]
   c. 決議：[決議內容]（藍色）
```

常見事項：
- 績效獎金計算及發放案
- 現金股利發放案

### 5.3 臨時動議（3.3）

```
3.3.1
   • 若無臨時動議：「此次會議沒有臨時動議提出，與會委員一致表示無需補充討論事項。」
   • 若有：依語音檔內容填寫
```

## 七、撰寫流程

1. **讀取語音檔 / 逐字稿**：擷取本次會議所有議題、決議內容、關鍵數字
2. **讀取會議附件**：核對財務數字、股權數據、法規文號等細節，確保精確
3. **讀取上次紀錄範本**：確認本次會議編號（N = 上次 + 1）、格式一致性
4. **填入固定段落**：會議時間、地點、與會人員（黑色）
5. **撰寫藍色內容**：依 3.1 → 3.2 → 3.3 → 3.4 順序逐一填入
6. **輸出 .docx 檔案**：
   - 藍色字體色碼：`#0070C0`
   - 字體：`Kaiti TC`
   - 層級縮排正確
   - 存放至 `/Users/jeffhsu/Desktop/Claude/投審會會議紀錄/`
   - 檔名格式：`圖靈一號投審會第N次會議紀錄_YYYYMMDD.docx`
