## 一、核心理念

**80/20 原則：**
- **80% 系統化**：100 分制評分、產業動能、估值、籌碼、技術指標
- **20% 主觀判斷**：消息面品質評估、時機微調、加減碼判斷

**交易紀律：**
- 每筆交易進場前必須填寫「進場理由」與「預設停損」
- 不符合進場條件清單的交易 → 標記為「紀律外」，納入勝率分析但獨立計算
- 週五收盤後固定回顧本週持倉

## 二、短線分析流程（Steps 1–7）

### Step 1 — 基本面 & 估值（20分）

> **短線邏輯：** 基本面不是 5–10 天漲跌的直接動力，但它決定「這股能不能放心賭」——EPS 為負、估值失控或月營收衰退，機構不會接棒。

**來源：** `five91.onrender.com/stock/{code}` 取得 EPS、PE、PB；MOPS 取最近月營收

| 欄位 | 說明 |
|------|------|
| TTM EPS | 正負與絕對值 |
| PE（TTM）| 與歷史 3 年區間比較百分位；無資料則與同業中位數比較 |
| PB（股價淨值比）| 判斷估值是否過熱 |
| 最近月營收 YoY | 短期基本面方向 |

**評分邏輯（20分）：**

| 條件 | 分數 |
|------|------|
| TTM EPS > 0（排除虧損股）| 5 |
| PE < 歷史 3 年 50 百分位（不算貴；無資料則 < 同業中位數得 3 分）| 5 |
| 最近月營收 YoY > 0%（短期基本面向好）| 5 |
| PB < 3 且 < 同業中位數（估值未失控）| 5 |

**文字總結（每次必輸出）：**
> 獲利能力：{正常 EPS X.X / ⚠️ 虧損}｜估值：{偏低 / 合理 / ⚠️ 偏高 PEx}｜月營收：{成長 YoY +X% / ⚠️ 衰退}
> → {結論 1 句：如「獲利穩健、估值合理，不構成進場障礙」/ 「估值偏高，需更強題材支撐」}

---

### Step 2 — 產業動能（40分）⭐ 核心評分

> **短線邏輯：** 5–10 天急漲，八成是題材輪動行情。產業動能決定「這波有沒有人一起推」，是評分權重最高的面向。

**數據來源：**
1. 從 Step 1 取得個股產業分類
2. WebSearch 查近 **5 個交易日**台灣對應產業指數或 ETF 漲跌幅
3. 查加權指數同期漲跌幅，計算產業 RS = 產業漲幅 − 大盤漲幅
4. 計算個股近 5 日漲幅 − 產業近 5 日漲幅 = 個股超額 RS
5. WebSearch 查近 7 日催化劑新聞（訂單、政策、客戶、法說會）
6. 查對應產業 ETF 近 5 日成交量是否放大（vs 20日均量）

#### 2.1 題材催化劑品質（20分）

| 條件 | 分數 |
|------|------|
| 有明確、近 7 日內的催化劑（訂單公告、政策利多、客戶新聞、財報超預期）| 10 |
| 題材具**可延續性**（非一次性新聞；有下一步期待值）| 6 |
| 公司是題材中的**核心受惠者**（非沾邊概念，主要業務直接受益）| 4 |

#### 2.2 產業相對強度（12分）

| 條件 | 分數 |
|------|------|
| 近 5 日產業指數 / ETF 漲幅 > 加權指數（產業板塊正在輪動）| 7 |
| 個股近 5 日漲幅 > 所在產業（個股強於板塊）| 5 |

#### 2.3 資金輪動訊號（8分）

| 條件 | 分數 |
|------|------|
| 對應產業 ETF 近 5 日成交量 > 20 日均量（機構資金湧入板塊）| 4 |
| 近 7 日媒體 / 外資報告有正面提及該板塊（WebSearch 確認）| 4 |

> **強制降級條件：** 產業動能得分 < 20/40 → 最高建議為「🟡 觀望」

**文字總結（每次必輸出）：**
> 產業：{產業名稱}｜近 5 日產業 vs 大盤：{+X.X% / -X.X%}（{跑贏 / 落後}）
> 催化劑：{30字描述}（品質：{強 / 中 / 弱}）
> 延續性：{可延續 / 一次性新聞}｜ETF 量能：{放大 ✓ / 未確認}
> → {結論}

---

### Step 3 — 技術面（20分）

> **短線邏輯：** 真正有效的是：**個股比大盤強嗎？（RS）+ 突破整理區間了嗎？（BB）+ 有量支撐嗎？（Vol）+ 有沒有在追高危險區？（RSI）**

**數據取得：** TWSE `STOCK_DAY` API，取近 **120** 個交易日（6 個月）

#### 3.1 相對強弱（RS）
```
個股 RS10 = (今收 − 10日前收) / 10日前收 × 100
大盤 RS10 = (加權指數今值 − 10日前值) / 10日前值 × 100
超額 RS = 個股 RS10 − 大盤 RS10
```

#### 3.2 布林通道（20日, 2σ）

| 位置 | 解讀 |
|------|------|
| 中軌上方且 < 上軌（50–100%）| 多頭健康區，最佳進場帶 |
| > 上軌（> 100%）| ⚠️ 超買，追高風險 |
| 中軌下方（< 50%）| 弱勢，不宜進 |
| < 下軌（< 0%）| 超賣，等反彈確認再入 |

#### 3.3 量能確認
- 近 5 日均量 / 近 20 日均量：> 1.2 = 放量 / < 0.8 = 縮量

#### 3.4 RSI（14日）
- 45–72：健康多頭區，計分
- > 75：⚠️ 追高危險，不計分且加警示

#### 3.5 ATR（10日）——動態停損停利
```
停損 = 1.5 × ATR(10)，上限 8%
停利 = 3 × ATR(10)，無上限
R/R = 停利% ÷ 停損%
```

**技術面評分邏輯（20分）：**

| 條件 | 分數 |
|------|------|
| 個股近 10 日超額 RS > 0 | 6 |
| 現價 > BB 中軌 且 ≤ BB 上軌 | 6 |
| 近 5 日均量 > 近 20 日均量 | 5 |
| RSI(14) 在 45–72 之間 | 3 |

**文字總結（每次必輸出）：**
> 相對強弱：近 10 日超額 RS {+X.X% / -X.X%}
> 布林位置：{X}%（{健康多頭區 / ⚠️ 突破上軌 / 中軌下方弱勢}），上軌 {X} / 中軌 {X} / 下軌 {X}
> 量能比：5日/20日 = {X}（{放量 / 縮量}）｜ATR(10) = {X}
> RSI(14)：{X}（{健康 / ⚠️ 超買}）
> → {結論}

### Step 4 — 籌碼面（20分）

> **短線邏輯：** **5 日累計**才能反映主力持續佈局意圖。外資 + 投信方向一致性比單日數字重要。

**加分旗標（不另計分，文字標記）：**
- 外資連買 ≥ 3 日 → `⭐ 外資趨勢性佈局`
- 外資 + 投信 5 日均正 → `🔥 雙法人共振`
- 外資 5 日 > +5,000 張 → `💰 大額外資流入`

**評分邏輯（20分）：**

| 條件 | 分數 |
|------|------|
| 外資 5 日累計淨買超 > 0 | 8 |
| 投信 5 日累計淨買超 > 0 | 7 |
| 三大法人 5 日合計 > 0 | 5 |

> **強制降級：** 外資 5 日累計 < −2,000 張 → 加「⚠️ 外資持續賣超」警示

**文字總結（每次必輸出）：**
> 外資 5 日累計：{+X,XXX / -X,XXX}張（{連續買/賣 N 日}）{旗標}
> 投信 5 日累計：{+X,XXX / -X,XXX}張
> 三大法人合計：{+X,XXX / -X,XXX}張
> → {結論}

---

### Step 5 — 100 分制綜合評分

| 面向 | 滿分 | 子項目 |
|------|------|--------|
| 基本面 & 估值 | 20 | EPS正向5 + PE位置5 + 月營收YoY5 + PB合理5 |
| 產業動能 ⭐ | 40 | 題材品質20 + 產業RS12 + 資金輪動8 |
| 技術面 | 20 | 超額RS6 + BB位置6 + 量能5 + RSI3 |
| 籌碼面 | 20 | 外資5日8 + 投信5日7 + 三大法人合計5 |
| **合計** | **100** | |

**進場判斷門檻：**

| 分數 | 建議 |
|------|------|
| 75+ | 🟢 建議進場 |
| 55–74 | 🟡 觀望 |
| < 55 | 🔴 不建議 |

**分級罰分制度：**

| 類型 | 條件 | 處理方式 |
|------|------|---------|
| 致命（封頂型）| TTM EPS ≤ 0 | 封頂 54 分 |
| 致命（封頂型）| 產業動能 < 20/40 | 封頂 54 分 |
| 扣分型 | RSI > 75（超買）| −10 分 |
| 扣分型 | 外資 5 日 < −2,000 張 | −8 分 |
| 扣分型 | 融資 5 日增加 > 3,000 張 | −5 分 |

邏輯：先算原始分（4 維度加總）→ 若有封頂型條件 `min(原始, 54)`；否則 `max(原始 − 扣分, 0)`

---

### Step 6 — ATR(10) 動態停損停利計算

```
停損% = min(1.5 × ATR(10) / 收盤價 × 100, 8%)
停損價 = 收盤價 × (1 - 停損%)
停利% = 3 × ATR(10) / 收盤價 × 100（無上限）
停利價 = 收盤價 × (1 + 停利%)
R/R = 停利% ÷ 停損%（低波動股 ≈ 2:1，高波動股 > 2:1）
```

## 三、PDF 版面規格（v9 已驗證）

**PDF 輸出路徑：** `~/Desktop/Stock Analysis/Reports/SwingTrade/{代號}_{公司名稱}_短線分析_{YYYYMMDD}.pdf`
**格式：** matplotlib 酒紅金暗色主題，figsize=(22, 15)，DPI 150

| 名稱 | 色碼 | 用途 |
|------|------|------|
| BG | `#0d1117` | 頁面背景 |

| WINE | `#922B21` | 邊框、強調色 |

**右側面板**為單一 axes `gs[1:, 1]`，以 `transAxes` 座標手動繪製三區塊：
- 評分卡（y 0.86–0.99）：圓形總分 + 建議信號燈
- 評分表格（y 0.20–0.84）：維度名稱 / 得分 / 滿分 + 罰分行 + 合計行
- 交易參數（y 0.01–0.18）：停損價 / 停利價 / R:R

### 關鍵設定備忘
- **禁止 `bbox_inches='tight'`**：會導致元素重新定位產生重疊
- **右側面板必須用單一 axes**：多 subplot 在右欄必定重疊
- **圖表摘要只能放 `set_title()`**：放圖內任何位置都會與數據重疊
- **字型順序**：`PingFang HK` → `STHeiti` → `Arial Unicode MS`
