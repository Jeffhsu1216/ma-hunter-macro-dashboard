# AUM_Overview.md｜全客戶 AUM 彙總規範

> 當使用者輸入「客戶總覽」時載入此文件，整合所有客戶資產數據，產出一頁式 PDF 彙總報告。

---

## 一、觸發詞與前置作業

**觸發詞：** `客戶總覽`

**執行前置：**
1. 讀取 `ClientPortfolio.md` 第一節取得客戶清單（姓名、類型、初始投資額、入金日期）
2. 讀取 `ClientPortfolio.md` 第三節取得各客戶 Excel 路徑規則
3. 依路徑規則，逐一找到每位客戶最新日期的 Excel（`openpyxl data_only=True`，**唯讀**）
4. ⚠️ **嚴禁對任何客戶 Excel 進行任何寫入或修改**

---

## 二、資料擷取規則

從每位客戶的 `Investment Portfolio` 工作表讀取：

| 儲存格 | 內容 |
|--------|------|
| `I25` | 期初投資額（本期滾動起始值） |
| `I27` | 當前餘額（= AUM） |
| `I29` | 本期總損益 |

若某客戶 Excel 找不到：標注「⚠️ 檔案未找到」，跳過該客戶，不影響其他人計算。

---

## 三、計算邏輯

| 指標 | 計算方式 |
|------|---------|
| 各客戶 AUM | `I27` |
| AUM 佔比 | 各客戶 `I27` ÷ 所有客戶 `I27` 加總 × 100% |
| 本期報酬率 | `I29 ÷ I25 × 100%` |
| 累計報酬率 | `(I27 ÷ 初始投資額 − 1) × 100%` |
| 總 AUM | 所有客戶 `I27` 加總 |
| 加權平均本期報酬率 | `Σ(各客戶 I29) ÷ Σ(各客戶 I25) × 100%` |
| 加權平均累計報酬率 | `Σ(各客戶 I27 − 初始投資額) ÷ Σ(各客戶初始投資額) × 100%` |

> 若初始投資額標記為「待補」，累計報酬率欄顯示「—」，不計入加權平均分母。

## 四、PDF 報告規範

### 4.1 檔案命名與路徑

- 檔名：`客戶總覽_{YYYYMMDD}.pdf`
- 路徑：`~/Desktop/Clients/AUM_Overview/`
- 例：`客戶總覽_20260329.pdf`

### 4.2 主題配色（酒紅金 暗色主題，與個別報告一致）

| 元素 | 色碼 | 用途 |
|------|------|------|
| 背景 | `#1a0a0a` | 全頁底色 |
| 酒紅 | `#8B1A1A` | 標題列 |
| 金色 | `#C8A951` | 標題文字、分隔線 |
| 淺金 | `#D4BC7A` | 副標題、小節標題 |
| 暖白 | `#F5F0E8` | 正文文字 |
| 卡片底 | `#2A1515` | 資訊卡背景 |
| 卡片邊 | `#4A2525` | 資訊卡邊框 |
| 正值 | `#4CAF50` | 損益正值（綠） |
| 負值 | `#E74C3C` | 損益負值（紅） |
| 灰文 | `#9E8E7E` | 標籤、說明文字 |

### 4.3 版面結構

使用 `matplotlib` + `GridSpec(4, 2, height_ratios=[0.07, 0.12, 0.45, 0.36])`

| 區域 | 內容 |
|------|------|
| **第一列（全寬）** | 酒紅標題列：`全客戶 AUM 彙總報告`，右側：`{今日日期} \| 客戶數：{N} 位` |
| **第二列（全寬）** | 4 張 KPI 卡片：總 AUM、客戶數、加權本期報酬率、加權累計報酬率 |
| **第三列左** | AUM 圓餅圖（各客戶佔比，含客戶名與金額標注） |
| **第三列右** | 本期報酬率雙橫條圖（左酒紅 = 本期，右金色 = 累計）依報酬高低排序 |
| **第四列（全寬）** | 客戶明細卡（每位客戶一格，共 6 格），各卡顯示：客戶名、類型、AUM、本期報酬率、累計報酬率 |
| **頁尾** | `資料截至各客戶 Excel 最新日期 \| 製表日期 {今日}` |

### 4.4 客戶明細卡規格

- 底色：`#2A1515`
- 頂部裝飾線：累計報酬率正值 → 綠色（`#4CAF50`）；負值 → 紅色（`#E74C3C`）；待補 → 金色（`#C8A951`）
- 顯示欄位：
  - 客戶名（大字，暖白）
  - 類型標籤（Fee-Based / Non-Fee，灰文）
  - AUM（金色，千分位）
  - 本期報酬率（正綠負紅）
  - 累計報酬率（正綠負紅，待補顯示「—」）

### 4.5 技術規範

- 語言：Python（`openpyxl` + `matplotlib`），**不呼叫 xlsx skill**
- 圖檔 DPI：150
- 尺寸：13.5 × 9.5 吋
- 字體：`Heiti TC`（確保中文顯示正常）
