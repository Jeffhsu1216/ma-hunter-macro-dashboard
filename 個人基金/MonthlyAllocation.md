## 三、配置邏輯（Gap Analysis）

### 核心原則：買齊 N 支 > Equal Weight
> 選出的股票儘量全部買到，比等權分配更重要。

### 3.1 計算步驟

1. **彙整持倉** — 讀取各客戶 Excel，聚合同代號多列（如靜怡昶昕兩列合併）
2. **計算可投資額** — `total_investable = 選股持倉市值合計 + 現金`（非選股持倉如矽格不計入）
3. **目標值** — `target = total_investable / N`（N = 選中股票數）
4. **差額分析** — 每支股票 `gap = target - 現持倉市值`
5. **買入建議** — gap > 0 才買，依零股規則計算張數/股數

### 3.2 零股規則

| 股價 | 買法 |
|------|------|
| >= 500 元/股 | 可買零股：`shares = floor(gap / price)`，再拆 `lots, odd = divmod(shares, 1000)` |
| < 500 元/股 | 只買整張：`lots = floor(gap / (price * 1000))`，odd = 0 |

**強制 1 張規則：** 若 <500 元股票 gap > 0 且「目前未持有」且算出 lots = 0 → 強制 lots = 1

### 3.3 跳過條件（資金不足不配置）

三個條件**同時成立**才跳過：
1. `total_buy > cash`（買入總額超過現金）
2. `shortfall > cash * 3`（缺口超過現金 3 倍）
3. `cash < min(underweight <500 股票的 1 張成本)`（連最便宜的一張都買不起）

→ 該客戶顯示「本月暫不配置」

## 四、PDF 輸出

### 輸出路徑
`~/Desktop/Clients/AUM_Overview/月選股下單建議_YYYYMM_含持倉_YYYYMMDD.pdf`

### 第一頁：持倉分析（2×2 四客戶卡片）

每張卡片包含：
- 標題：客戶名
- 副標題：現金 | 持股市值 | 可投資 | 目標/支
- 表格欄位：股票、現持股、現值、目標、差額、建議買入、成本
- 合計買入 / 剩餘現金
- 非選股持倉（如矽格）另行標示
- 跳過客戶：紅框顯示「本月暫不配置」+ 原因

### 第二頁：下單總結

- 僅列有下單的帳戶（跳過的不列）
- 每帳戶：深藍色標題欄 + 下單筆數/合計/剩餘
- 表格欄位：股票代號、股票名稱、股價、買入方式（整張/零股）、股數、預估成本
- 股數顯示：`1張`、`2張`、`294股`（不加括號）
- 全帳戶合計下單金額（綠色框）
- 底部註記跳過客戶

### 配色

| 元素 | 色碼 |
|------|------|
| 標題/表頭背景 | `#1B2A4A` |
| 買入建議/成本 | `#1A7A3A` |
| 已達標 | `#888888` |
| 跳過框 | `#CC3333` |
| 交替行背景 | `#F4F6F9` |
| 剩餘現金偏低 | `#D4760A` |

### 字體
`/System/Library/Fonts/STHeiti Medium.ttc`（中文）

## 五、執行流程

1. 讀取月選股 Excel → 掃 N 欄 = `"是"` → 取代號、名稱、收盤價
2. 逐一讀取 4 位客戶 Excel → 取 C25 現金、Row 4–19 持倉（唯讀）
3. 各客戶執行 gap analysis → 計算買入建議
4. 判斷跳過條件
5. 生成 PDF（第一頁持倉分析 + 第二頁下單總結）
6. 嚴禁對客戶 Excel 進行任何寫入或修改
